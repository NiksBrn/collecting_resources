# Cherry Beer

Бот для прохождения текстового данжена. Читает карту из файла, обходит комнаты, собирает ресурсы и возвращается на выход с максимальным счётом.

## Структура

```
.
├── main.cpp
├── headers/
│   ├── game.hpp        # игровая логика и состояние
│   ├── parser.hpp      # парсер входного файла
│   ├── bot.hpp         # интерфейс IBot
│   └── alice_bot.hpp   # реализация бота
└── src/
    ├── game.cpp
    ├── parser.cpp
    └── alice_bot.cpp
```

## Формат входного файла

```
N                                  # кол-во комнат 
id соседи [iron gold gems exp]     # N+1 строк (включая комнату 0 — выход)
...
M target                           # кол-во еды и таргет-ресурс
```

Пример:
```
3
0 1,2
1 0,2 5 3 10 0
2 0,1 0 0 0 2
3 2 1 0 0 0
4 iron
```

## Сборка

Требования: `cmake >= 3.10`, `g++ c++17`

```bash
mkdir build && cd build
cmake ..
cmake --build .
```

## Запуск

```bash
./build/cherry_beer input.txt
```

Результат записывается в `result.txt`.

## Алгоритм бота

1. **Исследование** — обходит непосещённые комнаты, приоритет соседям с меньшим номером. Тратит половину еды.
2. **Сбор** — в каждой новой комнате берёт самый ценный ресурс (первый бесплатно).
3. **Возврат** — кратчайшим путём через посещённые комнаты к выходу, по пути собирает ресурсы на оставшуюся еду.
